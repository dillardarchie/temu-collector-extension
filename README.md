# Temu Batch Collector

Chrome Extension: Auto batch collect Temu products to Miaoshou ERP.

## Features
- Auto click Miaoshou "Collect" button across all Temu tabs
- Configurable concurrency (5-20)
- Hotkeys: Ctrl+Shift+C (batch collect) / Ctrl+Shift+X (close collected tabs)
- Smart wait: auto-detect collection completion status

## For SHEIN Half-Hosted Sellers
Collect products from Temu and import to SHEIN platform.

## Install
1. Open chrome://extensions/
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select this repo folder

## Tech
- Manifest V3
- Supports earth-wxt-button Web Component + watermark character filtering
